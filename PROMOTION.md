# Ternimal - 高性能独立终端模拟器

## 产品简介

**Ternimal** 是一款基于现代 Web 技术构建的独立终端模拟器，从 Visual Studio Code 的官方代码库中提取成熟的终端架构模式，为您带来专业级的终端体验。

---

## 核心卖点

### 1. VS Code 级终端体验
- 直接复刻 VS Code 的终端架构设计
- 经过数百万开发者验证的稳定性和性能
- 专业级的终端渲染引擎

### 2. 硬件加速渲染
- **WebGL GPU 加速**：流畅处理大量输出
- **智能降级机制**：自动切换至 DOM 渲染确保兼容性
- **Unicode 11 完整支持**：完美显示各类特殊字符
- **真彩色支持**：256色及全色彩渲染

### 3. 多标签高效管理
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+T` | 新建标签页 |
| `Ctrl+W` | 关闭当前标签 |
| `Ctrl+Tab` | 切换到下一标签 |
| `Ctrl+Shift+Tab` | 切换到上一标签 |

### 4. 开发者友好设计
- **内置搜索**：`Ctrl+Shift+F` 快速查找终端内容
- **主题切换**：`Ctrl+Shift+L` 一键切换明暗主题
- **智能回滚**：5000 行历史记录缓冲区
- **右键即贴**：便捷的复制粘贴操作

### 5. 跨平台原生支持
- **Windows**：PowerShell/CMD，原生 ConPTY 支持
- **macOS**：默认 Shell，POSIX PTY 标准接口
- **Linux**：全功能终端，兼容各类发行版

## 技术架构

```
┌─────────────────────────────────────────┐
│           前端界面 (Electron)            │
│  ┌─────────────┐    ┌────────────────┐  │
│  │  xterm.js   │◄──►│   WebGL/DOM    │  │
│  │  终端渲染   │    │   渲染引擎      │  │
│  └──────┬──────┘    └────────────────┘  │
└─────────┼───────────────────────────────┘
          │ IPC 通信
┌─────────┼───────────────────────────────┐
│         ▼      后端进程 (Node.js)        │
│  ┌─────────────┐    ┌────────────────┐  │
│  │  node-pty   │◄──►│   伪终端管理    │  │
│  │  PTY 进程   │    │   生命周期控制  │  │
│  └─────────────┘    └────────────────┘  │
└─────────────────────────────────────────┘
```

### 核心技术栈
- **xterm.js**：业界领先的终端组件
- **node-pty**：跨平台伪终端绑定
- **Electron**：桌面应用框架

## 为什么选择 Ternimal？

| 对比项 | Ternimal | 系统自带终端 | 其他第三方终端 |
|--------|----------|--------------|----------------|
| 渲染性能 | WebGL 加速 | 普通 | 参差不齐 |
| 多标签支持 | 原生支持 | 部分支持 | 需要插件 |
| 主题切换 | 一键切换 | 有限 | 配置复杂 |
| 搜索功能 | 内置支持 | 多数不支持 | 需额外安装 |
| VS Code 体验 | 完全一致 | 无 | 无 |
| 跨平台一致性 | 完全一致 | 不一致 | 不一致 |

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/eathard/vscode-ternimal.git
cd vscode-ternimal

# 2. 安装依赖
npm install

# 3. 启动应用
npm run dev

# 4. 打包安装程序（可选）
npm run build
npm run pack
```
